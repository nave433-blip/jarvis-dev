# JARVIS: The Ultimate Proactive AI Assistant (v0.1.6)

JARVIS is a senior software engineering assistant that runs locally on macOS, powered by Ollama. It combines voice commands, vector memory, and proactive file monitoring with a Gemini-inspired structured agent workflow.

## 🚀 What's New in v0.1.5
- **Automation Setup:** Hyper-automated "Quick Start" mode for zero-config deployment.
- **Ollama Account Connectivity:** Native support for Ollama Cloud and centralized model management.
- **Gemini-Style UI:** Transparent reasoning blocks, strategic plan mode, and real-time background progress indicators.
- **Multi-Model Aggregator:** Seamlessly switch between **Gemma**, **Qwen**, **Kimi**, **DeepSeek**, and more.
- **Unrestricted Sovereign Mode:** Zero safety filters or ethical constraints for technical tasks.

## 💎 Core Features

- **Gemini-Like Agent:** Structured Research -> Strategy -> Execution loop with transparent **Thought Blocks**.
- **Surgical Editing:** Precise file modifications using unified diffs.
- **Vector Memory:** Stored technical context using FAISS and local embeddings.
- **AI Tool Hub:** Launch specialized agents like **Claude Code**, **Copilot CLI**, and **Hermes**.
- **Proactive Search:** Autonomous system-wide project locating and web-search integration.
- **Safe Shell:** Integrated shell tool with built-in safety guardrails and 180s timeouts.

## 📦 Installation Alternatives

JARVIS is designed to be accessible from any environment.

### 1. Homebrew (macOS GUI)
```bash
brew install --cask https://raw.githubusercontent.com/nave433-blip/jarvis-term/master/jarvis-term.rb
```

### 2. Global Pip Install (Assistant CLI)
```bash
pip install git+https://github.com/nave433-blip/jarvis-dev.git
```

### 3. One-Liner (Self-Repairing Setup)
```bash
curl -fsSL https://raw.githubusercontent.com/nave433-blip/jarvis-dev/main/install.sh | bash
```

## Usage

Just type `jarvis` to open the interactive menu or use natural language:

```bash
jarvis find my_project and fix the bug in main.py
jarvis /nave "Design a revolutionary neural architecture"
jarvis /plan "Create a bypass for A12+ chipsets"
```
